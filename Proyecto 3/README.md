## Prerequisitos de la aplicacion
- Tener MAUI .NET 10.0.X
- Tener Android Studio
- Tener OpenJDK 21.0.8 
- Tener Visual Studio Code 
- Tener git 2.X.X

## Dependencias
- ´dotnet add package CommunityToolkit.Mvvm´
- ´dotnet add package sqlite-net-pcl´
- ´dotnet add package SQLitePCLRaw.bundle_green´

## Instrucciones de ejecución
- Clonar el repositorio (git clone https://github.com/Neichoo/AppMoviles-INF320-202601-G2.git)
- Abrir Android Studio e iniciar un telefono con las siguientes caracteristicas:
    - Medium Phone.
    - API 36.0
    - Google APIs
- Abrir el Proyecto 3 en Visual Studio Code y darle correr proyecto desde "MyTest.csproj"
